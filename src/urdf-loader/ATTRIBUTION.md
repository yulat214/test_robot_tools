# URDF Loader Attribution

The files in this directory are vendored from:

**Source:** https://github.com/gkjohnson/urdf-loaders  
**Author:** Garrett Johnson  
**License:** Apache 2.0

## Files

- `URDFLoader.js` — Core URDF parser and Three.js scene builder
- `URDFClasses.js` — Robot/Joint/Link object definitions
- `URDFClasses.d.ts` — TypeScript type declarations
- `URDFDragControls.js` — Interactive joint drag controls
- `urdf-viewer-element.js` — Web component wrapper
- `urdf-manipulator-element.js` — Manipulator web component

## Notes

These files were copied and adapted for ESM/Vite compatibility.
Please check the upstream repository for the latest version before updating.
